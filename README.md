# ssuaring

this is our sharing app

### 1. Clone

1. git clone https://github.com/rat8397/ssuaring
2. prisma.yml 파일 넣기

### 2. Run

#### npm 일 경우

1. npm i : 노드 패키지 설치
2. npm run npm-prisma : generated 파일 생성 (deploy 안되는 오류 무시)
3. npm run dev : 서버 실행

#### yarn 일 경우

1. yarn: 패키지 설치
2. yarn prisma: generated 파일 생성
3. yarn dev: 서버 실행
